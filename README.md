# 银行支行绩效考核管理系统

## 项目简介

基于 Spring Boot + Vue.js 的前后端分离企业级管理系统，用于银行支行的绩效考核管理。系统实现了用户管理、部门管理、业务管理、绩效考核等核心功能，采用 RESTful API 设计风格，具备完善的权限管理和数据缓存机制。

**开发周期：** 个人独立开发  
**代码规模：** 约 10,000+ 行  
**项目亮点：** 前后端分离、JWT 认证、Redis 缓存、响应式数据访问

## 技术实现

### 后端实现要点

1. **RESTful API 设计**
   - 统一的请求响应格式
   - HTTP 状态码规范使用
   - 资源路由命名规范

2. **认证与授权**
   - JWT Token 机制实现无状态认证
   - 自定义拦截器验证 Token 有效性
   - 基于注解的权限控制 `@Auth`

3. **数据访问优化**
   - Spring Data JPA 简化 CRUD 操作
   - Redis 响应式缓存提升查询性能
   - Druid 连接池管理数据库连接

4. **异常处理**
   - 全局异常处理器 `@RestControllerAdvice`
   - 自定义业务异常体系
   - 统一错误响应格式

5. **数据验证**
   - Hibernate Validator 参数校验
   - 自定义校验注解
   - 业务逻辑校验分离

### 前端实现要点

1. **组件化开发**
   - 全局组件：分页、列表、详情弹窗
   - 业务组件：各模块功能组件
   - 组件通信：Props、Events、Vuex

2. **状态管理**
   - Vuex 模块化管理各业务状态
   - Actions 处理异步操作
   - Mutations 同步更新状态

3. **路由管理**
   - 路由懒加载优化首屏加载
   - 路由守卫实现登录验证
   - 动态路由配置

4. **网络请求**
   - Axios 拦截器统一处理请求/响应
   - Token 自动注入
   - 错误统一提示

5. **UI/UX 优化**
   - Element UI 组件库快速开发
   - 响应式布局适配
   - Loading 状态反馈
   - 表单验证提示

## 项目运行

### 后端启动

```bash
cd hsfbs
mvn clean install
mvn spring-boot:run
```

后端服务默认运行在 `http://localhost:8080`

### 前端启动

```bash
cd hsfbs-web
npm install
npm run serve
```

前端应用默认运行在 `http://localhost:8081`
